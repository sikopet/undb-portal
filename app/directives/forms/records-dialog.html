Understanding Bio-Bridge Records

UNDB records can be created by any user with a CBD account but must be approved by the CBD Secretariat before they are published. Any user can suggest edits to any published record.

UNDB records can reference other UNDB records. If a record you are creating needs to reference a record not already available you have to save your current record and create the reference record. Once the reference record is created you can edit the saved draft and select your new reference record.

UNDB records are presented on your dashboard as drafts, requests pending approval or as published records.


  1. Only you can see the record.

  2. You have submitted your record and it is pending approval.

  3. Everyone can see the record.
